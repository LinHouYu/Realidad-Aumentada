# 📱 Unity AR Prototype

这是一个使用 Unity + AR Foundation 构建的增强现实原型项目，支持图像识别、平面检测、3D 对象生成与交互、以及基础 UI 控制。

## 🚀 功能概览

- ✅ 平面检测（AR Plane Detection）
- ✅ 点击生成 3D 对象（Cube/Sphere）
- ✅ 图像识别（Image Tracking）
- ✅ 人脸检测（Face Tracking，设备支持时）
- ✅ UI 控件交互（按钮切换模型、滑块调整大小）

## 🧩 技术栈

- Unity 2021.3+（推荐 LTS）
- AR Foundation
- ARCore XR Plugin（Android） / ARKit XR Plugin（iOS）
- XR Interaction Toolkit（用于拖动/旋转/缩放）
- Input System（用于屏幕交互）
